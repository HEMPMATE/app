## Secure Online Examination System

Final presentation-ready ITS320 capstone prototype.

## Included
- `frontend/index.html` — GitHub Pages ready single-file app.
- `backend/` — Node/Express backend starter files.
- `database/schema.sql` — MySQL schema with relationships.

## Demo
No usernames or passwords are displayed on the page. Use **Create Account** to register a student, then log in and start a timed exam.

## GitHub Pages
Upload `frontend/index.html` as `index.html`, then enable GitHub Pages.

## Backend security design
JWT authentication, bcrypt password hashing, role-based access control, Helmet, rate limiting, input validation, and prepared MySQL statements to prevent SQL injection.
